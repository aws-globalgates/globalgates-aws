5

시멘틱 캐싱 — RAG는 score_threshold = 0.1 (보수적)

llm.md §1-9, §1-10, §5-6

일반 챗봇은 0.35~0.4지만 RAG는 사실성이 중요 → 0.1로 낮춰 잘못된 캐시 매칭을 차단. 그래도 의미가 거의 같은 질문은 즉답으로 가른다.

HIT 케이스 ~50ms

  1. "단가 얼마로 정해졌어?" → 임베딩
  2. FAISS-on-Redis similarity_search(k=1)
  3. score 0.05 ≤ 0.1 → 매칭
  4. 저장된 answer를 char 단위로 yield (시각적 타이핑)
  5. OpenAI 호출 0건, 토큰 비용 0

MISS 케이스 ~600ms TTFT

  1. "이번 회의 결론 요약해줘" → 임베딩
  2. 유사도 부족 → 풀 체인 진입
  3. Retriever(k=4) → Prompt → LLM astream
  4. 토큰 누적하며 클라이언트로 yield
  5. 완료 시점에 Redis.from_texts(question, answer) 저장
  6. 두 번째 같은 질문은 HIT 케이스로 강등

운영 가드레일: ① 회의 ID/사용자 ID를 인덱스 prefix(cache:meeting:{id})로 분리 → 다른 회의 답변이 새어나오는 사고 방지. ② 개인정보 포함 답변 캐싱 금지 플래그 — 시스템 프롬프트가 "참가자 외 노출 금지"인 답변은 no_cache=True로 우회. ③ 요약본이 갱신되면 해당 prefix invalidate.

from contextlib import asynccontextmanager
from fastapi import FastAPI
from dotenv import load_dotenv
from langchain.globals import set_llm_cache
from langchain_community.cache import RedisSemanticCache
from langchain_huggingface import HuggingFaceEmbeddings
from langchain_openai import ChatOpenAI

from app.core.config import settings
from app.api.routes import chat, index

@asynccontextmanager
async def lifespan(app: FastAPI):
    load_dotenv()
    # RAG 정확도 우선 → score_threshold = 0.1 (llm.md §5-6)
    embeddings = HuggingFaceEmbeddings(
        model_name="jhgan/ko-sbert-nli",
        model_kwargs={"device": "cpu"},
        encode_kwargs={"normalize_embeddings": True},
    )
    set_llm_cache(RedisSemanticCache(
        redis_url=settings.REDIS_URL,
        embedding=embeddings,
        score_threshold=0.1,
    ))
    app.state.llm = ChatOpenAI(model_name="gpt-5.4-mini", temperature=0)
    app.state.embeddings = embeddings
    yield
    # shutdown

app = FastAPI(lifespan=lifespan)
app.include_router(chat.router, prefix="/chat")
app.include_router(index.router, prefix="/index")
② 8단계 RAG LCEL — astreamfastapi/app/services/rag_chain.py
from langchain_community.vectorstores import FAISS
from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnablePassthrough

PROMPT = PromptTemplate.from_template("""
귀하는 화상회의 요약본을 바탕으로 질문에 답변하는 분석 전문가입니다.
반드시 아래 #Context 안의 정보만 활용하십시오. 자료로 답변이 어렵다면 추측 없이
'제공된 정보로는 확인이 불가능하다'고 명시하십시오. 모든 응답은 한국어로 작성합니다.

#Context:
{context}

#Question:
{question}

#Answer:""")

def build_chain(faiss_dir: str, embeddings, llm):
    vs = FAISS.load_local(faiss_dir, embeddings, allow_dangerous_deserialization=True)
    retriever = vs.as_retriever(search_kwargs={"k": 4})
    return (
        {"context": retriever, "question": RunnablePassthrough()}
        | PROMPT
        | llm
        | StrOutputParser()
    )

async def stream_answer(chain, question: str):
    async for token in chain.astream(question):
        yield token
③ Stream + Semantic Cache (직접 구현 — llm.md §1-10)fastapi/app/services/semantic_stream.py
from langchain_community.vectorstores import Redis
import asyncio

INDEX_NAME = "vca_semantic_cache"
SCHEMA = {"text": [{"name": "question"}], "tag": [{"name": "answer"}]}

async def stream_with_cache(message, embeddings, redis_url, chain, threshold=0.1):
    try:
        vdb = Redis.from_existing_index(embeddings, index_name=INDEX_NAME,
                                        redis_url=redis_url, schema=SCHEMA)
        results = vdb.similarity_search_with_score(message, k=1)
    except Exception:
        results = []

    if results and results[0][1] <= threshold:
        cached = results[0][0].metadata["answer"]
        for ch in cached:
            await asyncio.sleep(0.01)
            yield ch
        return

    full = ""
    async for token in chain.astream(message):
        full += token
        yield token

    if full:
        Redis.from_texts(texts=[message], embedding=embeddings,
                         metadatas=[{"question": message, "answer": full}],
                         index_name=INDEX_NAME, redis_url=redis_url)
④ FastAPI SSE 라우트fastapi/app/api/routes/chat.py
from fastapi import APIRouter, Request
from sse_starlette.sse import EventSourceResponse

from app.schemas.chat import ChatRequest
from app.services.rag_chain import build_chain
from app.services.semantic_stream import stream_with_cache

router = APIRouter()

@router.post("/stream")
async def chat_stream(req: ChatRequest, request: Request):
    llm = request.app.state.llm
    embeddings = request.app.state.embeddings
    chain = build_chain(faiss_dir=f"./faiss/{req.meeting_id}",
                        embeddings=embeddings, llm=llm)

    async def event_gen():
        async for token in stream_with_cache(req.question, embeddings,
                                              "redis://localhost:6380", chain):
            yield {"event": "token", "data": token}
        yield {"event": "done", "data": ""}

    return EventSourceResponse(event_gen())

async function askWithRag(meetingId, question, bubbleEl) {
  const res = await fetch("/api/video-ai/chat/stream", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ meetingId, question }),
  });
  const reader = res.body.getReader();
  const dec = new TextDecoder();
  let buffer = "";
  while (true) {
    const { value, done } = await reader.read();
    if (done) break;
    buffer += dec.decode(value, { stream: true });
    for (const line of buffer.split("\n")) {
      if (line.startsWith("data:")) {
        const token = line.slice(5).trim();
        bubbleEl.textContent += token;
      }
    }
    buffer = "";
  }
}
