학습/연동 화면

사용자가 말한 `/Users/yoonchan/Desktop/gb/ai/machine-learning/b_classifier` 작업 흐름에 맞춰 Jupyter에서 학습/검증하고 FastAPI에 심은 뒤 Spring Boot에서 호출합니다.

1 DB 추출 tbl_post, tbl_post_product, tbl_report, tbl_message, tbl_advertisement 등 CSV 생성
2 Jupyter 학습 전처리, 라벨링, train/validation/test 분리, 분류 모델 학습
3 검증 confusion matrix, macro F1, precision/recall, threshold 저장
4 FastAPI 모델별 `/classifiers/{name}/predict` 엔드포인트 제공
5 Spring 화면 작성 폼, 상품 등록, 채팅, 신고 큐, 광고 검토에 결과 표시
